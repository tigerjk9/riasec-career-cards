# PRD — RIASEC 직업 카드

## 1. 비전
> 학교 진로 부스의 표준이 되는 RIASEC 검사를 게이미피케이션 + AI 일러스트로 재해석한 직업카드.

## 2. 사용자
- 초·중·고 학생 (자기 흥미 유형 발견)
- 진로 교사 (수업 자료/포트폴리오 활용)

## 3. 사용자 시나리오
1. 「검사 시작」 → 18문항 1~5점 답변 (3분, 우측 레이더 차트가 함께 채워짐)
2. 결과: 6유형 점수 + 상위 2개 조합 + 매칭 직업 3개
3. **매칭 직업 1개 선택** + 이름·학교명 입력 + Gemini API 키
4. AI 일러스트 + 한국 명사(Hall of Fame) 자동 생성
5. 포스터형 직업카드 PNG 다운로드 → 인쇄

## 4. 기능 요구사항
- **F1.** 표준 RIASEC 18문항 (각 유형 3문항)
- **F2.** 6유형 점수 자동 합산 + TOP 2 조합 산출
- **F3.** 15개 조합 매트릭스 → 직업 3개 매칭 (단일 유형 fallback)
- **F4.** 실시간 레이더 차트 (답변마다 즉시 갱신)
- **F5.** 직업 1개 선택 + 이름·학교 입력
- **F6.** 포스터 카드: 색 헤더 / 이미지 / RIASEC 배지 / PERSONALITY STATS(별점) / CAREER MATCH / HALL OF FAME / 푸터(연도+학교)
- **F7.** PNG 다운로드 (html2canvas, scale 3, 폰트 로드 보장)

## 5. 비기능 요구사항
- BYOK — 키 외부 저장 X
- 단일 `index.html` + Vercel
- 카드 1장 생성 ~10초 (이미지 + 텍스트 병렬)
- 한국어 풀 라벨 필수 (영문 약자만 X)

## 6. 디자인 원칙
- 카카오톡 참고 카드 디자인 차용 (실제 운영 카드와 일관성)
- 6유형 색 매핑 (R 오렌지, I 시안, A 핑크, S 에메랄드, E 앰버, C 바이올렛)
- html2canvas 호환: 명시적 line-height, white-space, font-family fallback

## 7. 성공 지표
- 검사 완주율 (레이더 차트로 ↑)
- 카드 다운로드율
- 학교 진로 교사 채택률 (수업 자료)

## 8. 향후 (Out of scope)
- 36문항 정밀 모드
- 검사 결과 PDF 리포트
- 같은 유형 학생 매칭 (학급 활동 연계)
